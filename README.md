# 🚀 AI Manufacturing Product Generator

A multimodal AI application that generates **product descriptions** and **realistic images** from a single manufacturing idea.

---

## 🧠 Features
- 📝 AI-generated product description (using LLM)
- 🖼️ AI-generated product image
- ⚡ Fast response using Groq API
- 🎯 User-friendly UI with Streamlit

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **LLM:** Groq (LLaMA 3.1)  
- **Image Generation:** Stability AI  
- **Backend:** Python  
- **API Handling:** Requests  

---

## 🔄 Workflow

1. User enters a manufacturing/product idea  
2. Groq LLM generates product description  
3. Stability AI generates product image  
4. Results are displayed in UI
